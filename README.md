# Purist – A Gatsby starter

A clean, full-featured starter with built-in search, dark mode, and social sharing
– all for creating content with Gatsby.

## Getting Started

A guide on how to get started with Purist.

### Installation

Using the `gatsby-cli`:

```sh
gatsby new my-site https://github.com/sebsojeda/gatsby-starter-purist
```

Using `git`:

```sh
git clone git@github.com:sebsojeda/gatsby-starter-purist.git my-site

cd my-site

yarn
```

### Development & Build

Once the starter has been downloaded, you can begin development.

```sh
# Run localhost
yarn dev

# Production build
yarn build

# Clear all caches
yarn clean
```
